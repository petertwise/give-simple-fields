== Description ==

Simply adds a note or comment field to the Give donations plugin.

= Features =

Just one feature - adds a comment field to your donation forms.

== Installation ==

1. Upload the give-simple-note-field directory to the `/wp-content/plugins` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress.
1. There are no settings or further setup needed.
